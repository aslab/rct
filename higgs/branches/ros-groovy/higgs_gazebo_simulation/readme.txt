This package includes different nodes for creating a simulation environment for Gazebo.

config -- all config files for the different nodes (Gazebo:wg_world_kinect.sdf config file for the world)
launch -- all launch files (demo_navigation_amcl.launch -> Whole simulation demo_navigation_acml.launch -> To create a map of a simulation for navigation.
meshes -- meshes used in the model
models -- blend of the model
scripts -- Diferent scripts (bashim: to launch Gazebo easily)
src -- All source files
urdf -- urdf model definition for the pioneer

note: Execute gzclient on console to visualize simulation and choose correct initial pose.
