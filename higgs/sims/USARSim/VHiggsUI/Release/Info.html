Please set view point in the UTClient window!

  1. Press Alt+Tab to go to UTClient.
  2. Set view point.
  3. Left mouse button switch view point between robots.
  4. Right mouse button return to full view point.
  5. Press Alt+Tab again to return this GUI.
  6. Press OK button to continue.

NOTE: After you pressed the 'OK' button, UTClient will be automatically hid. If you want to show the UTClient window, please click the 'Show UT' button.
